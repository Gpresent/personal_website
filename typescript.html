Typescript

Typescript is a high level scripting language for web development that adds static types and type declarations to Javascript. Other than the typing system which Javascript doesn't support, there is no difference in how the languages are written. Typescript has the advantage of being stricter with its rules than Javascript, which means that accidental errors and bugs can be easier to spot.

When compiling a typescript page (the extension is .ts), the compiler will check for any objects that have been declared explicitly or implicitly (by assignment) to be one type and are later assigned an object of a different type. This overwriting of variables is a common error for those writing in Javascript, since variables are declared and overwritten without ever specifying a type.

Because a Typescript file must be a working javascript program with the addition of static types, the typescript compiler only has to remove type-related information and check for agreement of types. Then, the file is simply copied and renamed to {filename}.js.
